## Responsive Design

### Desktop (>768px)
- Two-column layout for messaging (groups list | messages)
- Grid layout for notes (3 columns)
- Full sidebar visible

### Mobile (<768px)
- Single column layout
- Collapsible sidebar
- Stack notes vertically
- Touch-friendly buttons

## User Flows

### Creating a Group (Admin)
1. Click "Messaging" in sidebar
2. Click "Manage Groups"
3. Click "New Group"
4. Enter name
5. Select members
6. Click "Save Group"

### Sending a Message
1. Click "Messaging" in sidebar
2. Select group from list
3. Type message
4. Click "Send" or press Enter

### Creating a Note (Admin)
1. Click "Notes" in sidebar
2. Click "New Note"
3. Enter title and content
4. Select visibility
5. Click "Save Note"

### Viewing Notes (All Users)
1. Click "Notes" in sidebar
2. View all accessible notes
3. Notes on dashboard show latest 3
